# Sound Files Directory

This directory contains optional sound effects for the Quiz Battle game.

## Expected Files:
- `correct.mp3` / `correct.wav` - Sound for correct answers
- `wrong.mp3` / `wrong.wav` - Sound for incorrect answers  
- `tick.mp3` / `tick.wav` - Timer tick sound

## Note:
Sound effects are optional. The game will work without them, but they enhance the user experience.
